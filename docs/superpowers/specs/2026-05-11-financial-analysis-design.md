# 公司年报核心数据分析系统 - 设计文档

## 1. 项目概述

### 1.1 项目背景
本系统旨在帮助企业对年报核心数据进行智能化分析，通过同行业对比发现改进方向，并对公司价值进行综合评价。该系统面向财务管培生-AI方向岗位的能力展示需求，重点体现AI技术在财务数据分析领域的应用。

### 1.2 目标用户
- **公司管理层**: CEO、CFO等高管查看整体财务状况和行业对比
- **财务团队**: 财务人员进行日常数据分析和报告生成
- **投资者**: 外部投资者了解公司价值和竞争力

### 1.3 核心价值主张
- 自动化数据采集与清洗，减少人工操作
- 智能分析引擎提供深度洞察
- 行业对标分析助力战略决策
- AI驱动的价值评估与预警机制

## 6. API接口设计

### 6.1 数据导入接口

#### 6.1.1 导入Excel数据

- **URL**: `/api/data/import/excel`
- **Method**: POST
- **Content-Type**: multipart/form-data
- **参数**:
  - `file`: Excel文件
  - `company_name`: 公司名称
  - `report_year`: 报告年份
- **响应**:
```json
{
  "success": true,
  "message": "数据导入成功",
  "data": {
    "rows_imported": 12,
    "company_id": 1
  }
}
```

### 6.2 数据分析接口

#### 6.2.1 获取公司财务指标

- **URL**: `/api/analysis/company/{company_id}`
- **Method**: GET
- **参数**:
  - `report_year`: 报告年份（可选）
- **响应**:
```json
{
  "success": true,
  "data": {
    "company_name": "示例公司",
    "report_year": 2023,
    "metrics": {
      "revenue": 100000000.00,
      "net_profit": 15000000.00,
      "gross_margin": 0.35,
      "net_margin": 0.15
    }
  }
}
```

#### 6.2.2 行业对比分析

- **URL**: `/api/analysis/compare/{company_id}`
- **Method**: GET
- **参数**:
  - `industry_name`: 行业名称
- **响应**:
```json
{
  "success": true,
  "data": {
    "company_name": "示例公司",
    "industry_name": "信息技术",
    "comparison": {
      "revenue": {"company": 100000000, "industry_avg": 85000000, "rank": "top 20%"},
      "gross_margin": {"company": 0.35, "industry_avg": 0.30, "rank": "top 15%"}
    },
    "suggestions": ["毛利率高于行业平均，建议保持", "营收增长潜力较大"]
  }
}
```

#### 6.2.3 公司价值评估

- **URL**: `/api/analysis/valuation/{company_id}`
- **Method**: GET
- **响应**:
```json
{
  "success": true,
  "data": {
    "company_name": "示例公司",
    "valuation_score": 85.5,
    "valuation_level": "A",
    "factors": {
      "financial_health": 90,
      "growth_potential": 82,
      "industry_position": 88,
      "risk_level": 78
    },
    "recommendations": ["建议关注成本控制", "拓展市场份额"]
  }
}
```

### 6.3 报告接口

#### 6.3.1 生成分析报告

- **URL**: `/api/report/generate/{company_id}`
- **Method**: POST
- **参数**:
  - `report_type`: 报告类型（full/summary）
- **响应**:
```json
{
```

## 7. AI能力设计

### 7.1 智能数据解析

- **技术**: NLP + 规则引擎
- **功能**: 自动识别PDF中的财务报表结构，提取关键数据
- **流程**: PDF解析 → 文本提取 → 表格识别 → 数据映射

### 7.2 异常检测模型

- **算法**: Isolation Forest / Z-score
- **功能**: 识别异常财务指标
- **触发条件**: 指标偏离行业均值超过2倍标准差

### 7.3 价值评估模型

- **算法**: 集成学习（随机森林/XGBoost）
- **特征维度**: 财务指标、行业地位、市场环境
- **输出**: 综合评分（0-100）

### 7.4 智能推荐引擎

- **技术**: 规则引擎 + 相似度匹配
- **功能**: 根据行业对比结果生成改进建议
- **输出**: 优先级排序的改进建议列表

## 9. 部署方案

### 9.1 开发环境

```bash
# 安装依赖
pip install -r requirements.txt

# 启动开发服务器
flask run --host=0.0.0.0 --port=5000
```

### 9.2 Docker部署

```dockerfile
FROM python:3.10-slim

WORKDIR /app

COPY requirements.txt .
RUN pip install -r requirements.txt

COPY . .

EXPOSE 5000

CMD ["flask", "run", "--host=0.0.0.0", "--port=5000"]
```

| 变量名 | 说明 | 默认值 |

## 附录：核心财务指标计算公式

| 指标 | 计算公式 |
|------|----------|
| 毛利率 | (营业收入 - 营业成本) / 营业收入 |
| 净利率 | 净利润 / 营业收入 |
| 资产负债率 | 总负债 / 总资产 |
| 流动比率 | 流动资产 / 流动负债 |
| 速动比率 | (流动资产 - 存货) / 流动负债 |
